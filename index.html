Habilidades

  • HTML

    Usado para crear la estructura de cada proyecto. Definimos la información que se presentará al usuario.

  • CSS

    Para dar estilos a la información que se está presentando, y para que las aplicaciones tengan un buen aspecto.

  • JavaScript

    El núcleo de toda aplicación web. Con él definimos el comportamiento de nuestras aplicaciones. Es el lenguaje de programación que sabe leer el navegador.

  • React.js

    Librería de JavaScript para creación de interfaces. Permite crear SPA y una mejor reutilización del código basado en componentes.

  • Node.js - Express.js

    Para usar JavaScript del lado del servidor. En mi caso, lo uso para crear las API REST de los proyectos y enviar datos a una Base de datos.

  • Webpack

    Con él podemos empaquetar varios tipos de archivos, para que al final, nos retorne un HTML, un CSS y un JS, que son los archivos que lee el navegador.

  • Bootstrap

    Nos otorga unos estilos ya definidos, que podemos usar de forma sencilla para no tener que crearlos desde cero.

  • Git (y GitHub)

    El sistema de control de versiones que utilizo es Git, y para almacenarlos en un repositorio, uso GitHub.
